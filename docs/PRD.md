# Planify — PRD v1 (résumé)

## Objectif
Plateforme SaaS pour piloter des prestations techniques/événementielles : devis → mission → facture → paiement.

## ICP (cible)
- Agences événementielles, équipes ops techniques, freelances structurés.

## Problèmes principaux
- Temps perdu sur devis/factures
- Manque de visibilité planning/logistique
- Suivi paiement/relance fragmenté

## KPI produit
- Activation : 1er devis en 24h
- Rétention : 1ère facture en 7j
- Temps cycle mission : création → paiement

## Scope MVP
- Onboarding + org + email code
- Clients / missions / devis / factures
- PDF + numérotation
- Paiements (manuel + Stripe)
- Basique logistique (inventaire/scan)

## Hors scope MVP
- IA avancée
- SSO
- Mobile native
