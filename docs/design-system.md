# Design System — Planify

## Tokens (CSS vars)
- Couleurs définies dans `apps/web/src/app/globals.css`
- Palette gris/blanc + accent vert

## Composants
- Button, Card, Badge, Table, Tabs, Input, Textarea
- États pris en charge : hover, disabled, loading

## États UX
- Loading : skeleton/texte “chargement…”
- Empty : message court + CTA
- Error : message + action possible

## Principes
- Densité modérée, lisibilité
- Tables alignées, ellipsis
- CTA primaire unique par écran
